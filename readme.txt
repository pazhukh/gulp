1. В нашій кореневій папці проекту занисткаємо шифт і праву кнопки миші -  заходимо в термінал
2. В терміналі пишемо :  npm init  (в папці зявляється package.json)
3. В терміналі пишемо:  npm i --save-dev gulp 
npm i --save-dev gulp-sass browser-sync gulp-autoprefixer

4. В кореневу папку запихаємо всі папки і файли що є у папці app
5. В кореневу папку додаємо файл gulpfile.js
